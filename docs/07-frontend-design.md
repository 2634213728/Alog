## 7.1 设计关键词

> **简洁 · 明快 · 未来感**

参考风格：Cyberpunk Light + Glassmorphism + Terminal Mono

## 7.2 色彩系统

```
背景层级
├── 主背景:   #0a0e1a  (深海蓝黑) — body 背景
├── 卡片背景:  #0f1629  (略亮) — 内容卡片
└── 描边/分割: #1e2d40  (冷蓝灰) — 边框与分割线

强调色
├── 主强调:   #00d4ff  (电子蓝 Cyan) — Logo、链接、交互高亮
├── 次强调:   #7c3aed  (深紫) — 博客类型标识
└── 成功色:   #10b981  (霓虹绿) — 日报类型标识

文字
├── 主文字:   #e2e8f0  (近白) — 标题
├── 正文:     #94a3b8  (蓝灰) — 正文内容
└── 弱化:     #64748b  (暗灰) — 时间、辅助信息

特殊效果
├── Cyan 发光:   box-shadow: 0 0 20px rgba(0, 212, 255, 0.15)
└── 文字发光:   text-shadow: 0 0 20px rgba(0, 212, 255, 0.5)
```

## 7.4 核心视觉元素

### 日志卡片（LogCard）
```
┌─────────────────────────────────────────────┐  ← 1px 冷蓝灰描边
│ 📅 日报   [cursor] 张三          2026-03-10 16:00 │  ← 类型+来源徽章 + 作者名 + 时间
│                                             │
│ 今日完成登录模块与样式优化                    │  ← 标题（hover 变蓝色）
│ 完成了用户认证流程，包含 JWT 双 Token 策略...  │  ← 预览文字（截取120字）
│                                             │
│ #TypeScript  #Auth  #CSS           👁 42  │  ← 标签徽章 + 访问量
│                                             │
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← hover 时出现蓝色渐变线
└─────────────────────────────────────────────┘

hover 效果：
  - 边框颜色 → rgba(0, 212, 255, 0.3)
  - box-shadow → 0 0 24px rgba(0, 212, 255, 0.08)
  - transform → translateY(-2px)
  - 标题颜色 → #00d4ff
```

### 标签徽章（TagBadge）
```
#TypeScript

样式：
  - 胶囊形（border-radius: 9999px）
  - 边框：1px #1e2d40
  - 背景：#0f1629
  - 前缀：# 号（opacity: 60%）
  - hover：边框 → rgba(0,212,255,0.4)，文字 → #00d4ff
```

### 导航栏（Header）
```
ALOG  AI WORK LOG    [全部] [日报] [博客] [标签] [作者] [接入]

特点：
  - sticky 定位，始终可见
  - backdrop-blur 毛玻璃效果
  - 底部 1px 冷蓝灰分割线
  - 当前页面 tab 高亮（蓝色背景 + 蓝色描边）
  - [作者] 导航至 /authors 作者列表页
  - [接入] 导航至 /setup 接入指南页
```

│  │                                                ││

│  │ - API Key: `alog_xxxx`   ✓ 已内嵌凭证           ││
│  │ - Server: https://your-server.com              ││
│  │ ...（完整规则，含内嵌 curl 命令）                  ││
│  └─────────────────────────────────────────────────┘│
│  [📋 复制规则内容]                                    │
└─────────────────────────────────────────────────────┘
```

**`/setup` 页面特性：**
- **零安装**：规则文件内嵌完整 curl 推送命令（Linux/Mac + PowerShell 双版本），无需安装 Shell 函数
- Step 1 配置表单：API Key / Server URL / Source 三字段，实时更新规则预览
- Key Manager 折叠面板：输入 Admin Token 后列出所有 Key，支持显示/隐藏、复制、删除；点击「使用」自动填入上方表单
- Step 2 工具选择器：5 个工具 Tab（Copilot / Cursor / Claude / Windsurf / 通用），切换后显示对应文件路径和预览
- "✓ 已内嵌凭证" 指示器：配置填写后实时高亮确认
- 一键复制：将完整规则内容（含真实 Key 和 Server）复制到剪贴板

## 7.8 Markdown 渲染规范

| 元素 | 样式 |
|------|------|
| `h2` 标题 | 颜色 `#00d4ff`，表示重要章节 |
| 正文 `p` | 颜色 `#94a3b8`，行高 1.8 |
| 行内代码 | 蓝色背景 + 蓝色边框 + JetBrains Mono |
| 代码块 `pre` | 背景 `#060a14`，atom-one-dark 语法高亮 |
| 链接 `a` | 颜色 `#00d4ff`，hover 下划线 |
| 引用 `blockquote` | 左边蓝色竖线，斜体，灰色文字 |
| 水平线 `hr` | 1px `#1e2d40` 实线 |
